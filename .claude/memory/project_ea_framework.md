## 策略框架（XAUUSD v1.3）

**逻辑层次：**
- Layer 1 — H4 趋势过滤：EMA50 > EMA200 + ADX > 25
- Layer 2 — H1 入场信号：Pin Bar 或 Engulfing，价格在 EMA40 附近
- Layer 3 — 风险管理：ATR×2.5 止损，1%风险/笔，1:2半平+ATR追踪
- Layer 4 — 熔断机制：回撤20%停开仓，30%强平，冷却21天
- Layer 5 — 新闻过滤：事件前60分钟/后30分钟停开仓

**参数设置：**

| 参数 | 值 |
|------|-----|
| H4 Fast EMA | 50 |
| H4 Slow EMA | 200 |
| H4 ADX Period/Min | 14 / 25 |
| H1 EMA Period | 40 |
| ATR Period | 14 |
| SL ATR Multiplier | 2.5 |
| TP RR | 2.0（半平） |
| Trail ATR Multiplier | 2.0 |
| Risk Per Trade | 1.0% |
| Max Unprotected | 2 |
| MaxEntrySlippage | 0.3×ATR |

## 已识别问题（待解决）

1. **入场信号质量差：** Pin Bar / Engulfing 在黄金H1信噪比低，假信号多
2. **止损机械：** ATR×2.5是固定距离，容易被大资金精准猎杀
3. **出场太早：** 固定1:2在趋势年份过早锁定利润
4. **无市场状态区分：** 趋势行情、避险行情、震荡行情用同一套参数
5. **风险管理静态：** 单笔1%不随策略近期表现动态调整

## 下一步优化方向（优先级排序）

**优先级1：加入日线市场状态分类**
- 目标：区分"宏观趋势期 / 避险驱动期 / 震荡期 / 高波动无方向期"
- 实现：日线ADX + H4 ATR相对均值 + 价格与日线EMA200的位置关系
- 预期效果：减少30~40%假信号

**优先级2：用FindSwingLow()替换ATR止损**
- 目标：止损放在结构位之外，而非机械ATR距离
- 实现：已有FindSwingLow() / FindSwingHigh()函数，未被使用
- 注意：只需改OpenTrade()里的sl_price计算

**优先级3：动态出场（ADX自适应追踪）**
- 目标：趋势强时放手让利润跑，趋势弱时收紧止损
- 实现：ADX持续上升→暂停追踪；ADX回落→收紧至1×ATR

**优先级4：流动性猎取入场信号（替换Pin Bar）**
- 目标：等大资金触发散户止损后再入场
- 逻辑：价格刺破前低后快速收回 → 假突破确认 → 入场
- 复杂度：中等，建议单独立项

## 关键判断结论

1. 15%年化概率约55%，20%约35%——在$10k~$50k规模下可实现
2. 不建议追求更高收益——激进配置在困难年份损失会抵消多年复利
3. 股票不比外汇/商品更容易——无杠杆、跳空风险、点差成本三重劣势
4. 股指期货/CFD可考虑加入——在MT5框架内比股票现货更适合趋势EA
5. 参数优化是陷阱——策略鲁棒性来自逻辑正确，不来自参数精确
6. 与顶级团队核心差距——在信号层（微结构/跨资产）和研究流程
